# Trojan Whore Website

This repository hosts the official website for Trojan Whore, a cybergrind / noisecore / goregrind / gorenoise one-man project.

The site showcases:

- Releases with embedded Bandcamp players and album artwork

- Videos including full-length uploads and Shorts from YouTube

- Photos of the project, artwork, and live shots

- Social links for Bandcamp, TikTok, Instagram, etc.
